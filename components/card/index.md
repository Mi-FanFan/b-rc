通用卡片容器。

## 何时使用

最基础的卡片容器，可承载文字、列表、图片、段落

## API

| 参数      | 说明                                     | 类型       | 默认值 |
|-----------|------------------------------------------|------------|--------|
| children     | 子组件                                 | React.ReactNode | 无     |
| bordered | 是否有边框  | boolean     | true    |
| loading | 是否是预加载的卡片  | boolean     | false    |
| className | 自定义样式                            | string | 无     |
