#B-RC 

B-RC Components build with [React](http://facebook.github.io/react).

B-RC Components design for [MiFanFan](http://www.mifanfan.cn) Desktop front-end projects

## Installation 

With [npm](http://npmjs.com)
 
 ```
 npm install b-rc --save
 ```
